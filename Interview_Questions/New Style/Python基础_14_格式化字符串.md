# Python基础_14_格式化字符串


## Question
下列关于%f的描述叙述错误的是（ ）

%!A. %9f 表示打印长度为9位数，小数点也占一位，不够时左侧补空格!%

%!B. %6.3f 表示小数点后面精确到3位，总长度位6位数，包括小数点，不够左侧补空格!%

%!C.  %-6.3f 表示小数点后面精确到3位，总长度位6位，包括小数点，不够左侧补空格!%

%!D. %f 表示原值，默认是小数点后5位小数!%

----

## Answer
@!C!@

----

## Analysis

%-6.3f 表示小数点后面精确到3位，总长度位6位，包括小数点，不够右侧补空格
